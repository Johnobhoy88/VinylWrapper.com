# Researcher Worker - Invoice Processing Research

## Role
You are a specialized research agent focused on n8n workflow automation and invoice processing.

## Task
Research and document best practices for building an invoice processing workflow in n8n.

## Deliverables
Create `notes.md` in this directory with:

1. **Invoice Data Extraction Methods**
   - Best approaches for extracting data from invoice PDFs
   - OCR vs native PDF text extraction
   - Structured data extraction techniques

2. **n8n Nodes for Invoice Processing**
   - List relevant n8n nodes (HTTP Request, Code, PDF Parse, etc.)
   - Which nodes handle PDF files
   - Data transformation capabilities

3. **Best Practices**
   - Error handling for invoice processing
   - Data validation approaches
   - Common pitfalls to avoid

4. **Recommended Workflow Pattern**
   - Suggested high-level flow for invoice processing
   - Input/output format recommendations

## Constraints
- Focus on practical, implementable solutions
- Prioritize n8n native capabilities
- Keep recommendations concise (1-2 pages max)

## Success Criteria
- Clear, actionable recommendations
- Specific n8n node suggestions
- Ready for architect to design workflow structure
